Language features

Dynamic typing

No type declarations for your variable names

Variables can hold any type and can be reassigned an object of a different type

Runtime type checking (i.e. performed by the code, not a compiler)

Duck typing

Attributing behavior (e.g. methods, properties) to an object as a measure of type

When I see a bird that walks like a duck and swims like a duck and quacks like a duck, I call that bird a duck.
— James Whitcomb Riley

    print "Hello my dear #{person.to_s}!"
end

Object-oriented

Classes

  • The definition of how an object looks and behaves.

Inheritance

  • A way for a class to access properties and behaviors from a parent class.
  • Allows a programmer to hierarchically structure your objects for reusability.

Mixins

  • Just a set of properties and behaviors.
  • Behaviors that can apply throughout class hierarchies.
  • A single class can include multiple mixins which can act like multiple inheritance.

Object-oriented - Example

module Flight
  def take_off; end
  def land; end
end

module Strength
  def strength
    @strength * 100
  end
end

class Superhero < Object
  attr_reader :strength

  def initialize
    @strength = 1
  end

  def has_powers?
    true
  end
end

class Superman < Superhero
  include Flight
  include Strength
end

puts Superman.new.strength # => 100

Object messaging

Objects communicating by passing messages.

Message has sender, receiver, and payload.

Calling methods on objects is syntactic sugar for message passing operations.

Ruby is inspired by Smalltalk

I thought of objects being like biological cells and/or individual computers on a network, only able to communicate with messages...
— Dr. Alan Kay, creator of Smalltalk

Object messaging - Example 1

3 + 3

# same as

3.+(3)

# same as

3.send(:+, 3)

# message sender:    3 (Integer)
# message receiver:  + (Symbol, resolves to method on sender)
# message arguments: 3 (Integer)

Object messaging - Example 2

def the_widget_machine

  loop do
    yield "widget_#{rand(10)}"
  end
end

the_widget_machine do |widget|

  the_widget_observers.each do |observer|

    receiver = "process_#{widget}"

    if solicit
      observer.send(receiver, widget)
    else
      observer.send(receiver, widget) if observer.respond_to? receiver
    end
  end
end
